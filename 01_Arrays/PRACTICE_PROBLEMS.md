## üí° Common Patterns to Master

1. **Two Pointers**
   - Left and right pointers
   - Examples: Two Sum (sorted), Container With Most Water

2. **Sliding Window**
   - Fixed or variable size window
   - Examples: Maximum Subarray, Sliding Window Maximum

3. **Hash Map/Set**
   - Fast lookups
   - Examples: Two Sum, Contains Duplicate

4. **Prefix/Postfix**
   - Cumulative calculations
   - Examples: Product Except Self, Maximum Subarray

5. **In-place Modification**
   - Modify array without extra space
   - Examples: Rotate Array, Set Matrix Zeroes

## üìù Interview Tips

1. **Always clarify:**
   - Can I modify the input array?
   - What if array is empty?
   - Are there duplicates?
   - What's the range of values?

2. **Think out loud:**
   - Explain brute force first
   - Then optimize
   - Discuss time/space complexity

3. **Test your solution:**
   - Edge cases (empty, single element)
   - Normal cases
   - Large inputs

4. **Optimize:**
   - Can we use hash map for O(1) lookup?
   - Can we use two pointers?
   - Can we do it in-place?
